# Prueba Interfell Ionic

Este proyecto fue generado con [IONIC] en su versión 3.

## Requisitos

- NPM versión 10.16
- Typescript 3.4.5

## Instalación

Comandos necesarios
```
git clone https://github.com/Ris3TwO/ioniclogin.git
npm i -g ionic@4 //De no tener instalado ionic
npm i
ionic serve
```
